## デフォルト設定（FORMAT:default）

Web小説投稿サイト（小説家になろう・カクヨム等）向けの標準設定。

---

### 基本禁則

**行頭禁則（行の先頭に置かない文字）：**
```
、。，．・：；？！゛゜´｀¨^ ̄＿ヽヾゞ〃仝々〆〇
―‐∥…‥～～∥｜〔〕〈〉《》「」『』【】
```

**行末禁則（行の末尾に置かない文字）：**
```
（〔〈《「『【
```

---

### 句読点・記号

| ルール | 内容 |
|---|---|
| 読点 | 全角「、」を使う（半角カンマ「,」は不可） |
| 句点 | 全角「。」を使う（半角ピリオド「.」は不可） |
| 感嘆符 | 全角「！」を使う。連続は「！！」まで |
| 疑問符 | 全角「？」を使う。連続は「？？」まで |
| 三点リーダ | 「……」（2つ続けて使用）。「...」「…」1つは不可 |
| ダッシュ | 「――」（2つ続けて使用）。「--」「─」1つは不可 |
| 括弧 | 「」（かぎ括弧）→ 会話文。（）→ 心の声・補足 |

---

### 会話文のルール

```
・会話は必ず「」で囲む
・会話末尾に。は置かない（「こんにちは」。× → 「こんにちは」○）
・例外：「〜だ。でも〜」のように会話内に複数文がある場合は。を含めてよい
・会話の地の文接続は改行で区切る
```

---

### 段落・改行

```
・段落の書き出しは全角スペース1字分の字下げ（一般的なweb小説に合わせる）
・会話文は独立した段落にする
・連続会話（掛け合い）は1行ずつ改行
・情景・地の文のブロックは2〜4文を目安に改行を入れる
```

---

### 文体

```
・一文の目安：全角40字以内（40字を超える文が3文連続したら分割を検討）
・体言止めは効果的な場所にのみ使う（連続は避ける）
・同じ語尾が3文以上連続しないようにする（「〜た。〜た。〜た。」は回避）
・同じ単語が1段落内に3回以上出たら言い換えを検討する
```

## FORMAT コード一覧

`QUILL_STATE` の `FORMAT` フィールドで指定する。

| コード | 説明 |
|---|---|
| `default` | 上記すべてを適用（標準） |
| `minimal` | 行頭・行末禁則のみ。句読点・文体チェックなし |
| `novel` | defaultに加え、一文30字以内・体言止め制限強化 |
| `free` | 禁則処理なし |

## 禁則処理の確認タイミング

1. シーン執筆時（PHASE_2 STEP 3）：執筆しながら適用する
2. 評価時（eval_engine.md 軸3「読みやすさ」）：違反箇所を指摘する
3. ユーザーが「読み返したい」「修正したい」と言ったとき
